Now, here are the instructions for running the Project:

To run the project, you need to have a virtual server installed on your computer. We recommend using XAMPP for this purpose. 

Once you have XAMPP running, follow these steps:

1. Extract the downloaded file.
2. Copy the main project folder.
3. Paste the folder in the 'htdocs' directory of XAMPP.
4. Next, you need to connect to the database:

Open your web browser and go to the URL "http://localhost/phpmyadmin/".
5. Click on the 'databases' tab.
6. Create a database named "schooligniter" and click on the 'import' tab.
7. Click on the 'browse' button and select the "schooligniter.sql" file located in the 'DATABASE FILE' folder.
8. Click on the 'Go' button.
9. Once the database is created, you can now run the system:

10. Open your web browser and go to the URL "http://localhost/School".
